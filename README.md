# SmallRPG
A small RPG Game written in Python

This game is a non-linear RPG.
You will be a warrior and start your advanture in an unknown place. 
There will be forests, rivers, and castles, so drawing out the map while playing is highly recommended. This game has multiple endings, all depends on how much you find out about the world. Also, each decision matters, and sometimes there are no turning back!

Enjoy the Game!
